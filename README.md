# case-poo-feadev

Esse repositório contém o arquivo "introducao_poo_fea_dev.ipynb", que faz parte da semana de nivelamento da FEA Dev.<br><br>
O arquivo contém a resolução de alguns exercícios que exploram aspectos de programação orientada a objetos (abstração, encapsulamento, herança e polimorfismo), bem como um projeto final de backtesting usando a biblioteca [Backtrader](https://www.backtrader.com/).
